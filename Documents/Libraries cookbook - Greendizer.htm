Introduction

The Greendizer libraries Cookbook

We built Greendizer libraries on top of our API to allow developers to create applications for Greendizer quickly and efficiently. The library is proposed in four languages: Python, Java, PHP, and C#. This guide will walk you through the libraries features and the ways you can integrate them in your projects.

Audience

This guide is meant for people familiar with our web applications as well as some basic OOP concepts.

We also strongly advice you to take a look at the Greendizer API documentation before diving into this guide.

Get Started

The Client Object

The Client object is the entry point to the library. You can either initiate a BuyerClient or a SellerClient depending on the context you are in. For more information about contexts, check the API documentation.

HTTP basic access authentication

To use the basic HTTP access authentication, you need to give your username and password as parameters to the client constructor.

  • Python
  • Php
  • Java
  • Javascript
from greendizer import BuyerClient, SellerClient


buyer_client = BuyerClient(username="jimi.hendrix@springshield.com", 
                           password="password")

seller_client = SellerClient(username="jimi.hendrix@springshield.com",
                             password="password")
require_once 'Greendizer/BuyerClient.php';
require_once 'Greendizer/SellerClient.php';

$buyer_client = new Greendizer_BuyerClient(
                        array(
                            'email' => "jimi.hendrix@springshield.com",
                            'password' => "password"
                        )
);

$seller_client = new Greendizer_SellerClient(
                        array(
                            'email' =>  "jimi.hendrix@springshield.com",
                            'password' => "password"
                        )
);
import com.greendizer.api.client.BuyerClient;
import com.greendizer.api.client.SellerClient;


BuyerClient buyerClient = new BuyerClient(
    "jimi.hendrix@springshield.com",
    "password"
);

SellerClient sellerClient = new SellerClient(
      "jimi.hendrix@springshield.com",
      "password"
);
//Not supported in JavaScript.

$seller_client = new Greendizer_SellerClient(array('token' => $oauth_token));

BuyerClient buyerClient = new BuyerClient(token);

SellerClient sellerClient = new SellerClient(token);

Containers

Containers are meant to be the way to access resources and collections. They expose methods to access resources by id or collections by query string or simply to get the count of resources.

Containers are resource attributes; they therefore allow chained calls. The following code snippet retrieves a given invoice . The emails() and invoices() methods return Container objects while the method getById() returns a resource.

  • Python
  • Php
  • Java
  • Javascript
from greendizer import BuyerClient


user = BuyerClient(oauth_token="oauth_token").user

#Retrieve the email address to which the invoice was sent
email = user.emails['1234567890']

#Retrieve the desired invoice using its ID
invoice = email.invoices['9891']
require_once 'Greendizer/BuyerClient.php';

$buyerClient = new Greendizer_BuyerClient(array('token' => $oauth_token));
$user = $buyerClient->getUser();

//Retrieve the email address to which the invoice was sent
$email = $user->emails()->getById('1234567890');

//Retrieve the desired invoice using its ID
$invoice = $email->invoices()->getById('9891');
import com.greendizer.api.resource.buyer.Buyer;
import com.greendizer.api.resource.buyer.Email;
import com.greendizer.api.resource.buyer.Invoice;

Buyer user = new BuyerClient("oauth_token").getUser();

//Retrieve the email address to which the email was sent
Email email = user.getEmails().getById("1234567890");

//Retrieve the desired invoice using its ID
Invoice invoice = email.getInvoices().get("9891");
application.init(function(state, error){
    if(state){
        var user = application.getUser();
        var email = user.getEmails().getItemByID("1234567890");
        
        //Retrieve the desired invoice using its ID.
        var invoice = email.getInvoices().getItemByID("9891", successFn, failureFn);
        
        /*
        * 'successFn' and 'failureFn' are optional callbacks.
        * If omitted, the resource will be retrieved from the local cache,
        * or otherwise instantiated.
        
        * You can always call the "load(successFn, failureFn)" method later on
        * if you wish to involve the server.
        */
    }
});

Collections

The libraries represent collections as objects that contain a list of resources. Collections are returned by the All property and search() method of containers.

To access the collection resources, you first need to populate it. The collections populate() method retrieves the resources contained in the collection.

  • Python
  • Php
  • Java
  • Javascript
from greendizer import BuyerClient


def get_all_invoices():
    '''
    Returns all the invoices sent to 
    jimi.hendrix@springshield.com
    '''
    user = BuyerClient(oauth_token='oauth_token').user
    email = user.emails['jimi.hendrix@springshield.com']

    #Instantiate a collection
    collection = email.invoices.all
    
    collection.populate()       #Calling the populate() method with no parameters
                                #will load the last 200 resources available. 

    #Access the retrieved invoices
    return collection.resources
require_once 'Greendizer/BuyerClient.php';

function get_all_invoices(){

    $buyerClient = new Greendizer_BuyerClient(array('token' => $oauth_token));
	$user = $buyerClient->getUser();
    $email = $user->emails()->getById('jimi.hendrix@springshield.com');
    
    //Instantiate a collection
    $collection = $email->invoices()->getAll();
    
    $collection->populate();    //Calling the populate() method with no parameters
                                   //will load the last 200 resources available. 
    
    //Access the retrieved invoices
    return $collection->getResources();
}
import com.greendizer.api.client.BuyerClient;
import com.greendizer.api.resource.buyer.Buyer;
import com.greendizer.api.resource.buyer.Email;
import com.greendizer.api.resource.buyer.Invoice;
import com.greendizer.api.dal.Collection;

/**
 * Returns all the invoices sent to
 * jimi.hendrix@springshield.com
 */
public Collection<Invoice> getAllInvoices() {
  Buyer buyer = new BuyerClient("oauth_token").getUser();
  Email email = buyer.getEmails().get("jimi.hendrix@springshield.com");

  //Instantiate a collection
  Collection<Invoice> collection = email.getInvoices().all();

  //Calling the populate() method with no parameters
  //will load the last 200 resources available
  collection.populate();

  //Access invoices as you would do through a java.util.List
  //as Collection implements this interface.
  return collection;
}
/*
* Returns all the invoices sent to a "jimi.hendrix@greendizer.com"
* @param {greendizer.oauth.BuyerApplication} buyerApplication Instance of a buyer application.
* @param {Function} callbackFn Callback function
* @returns {Array} Array of invoices. 
*/
function getAllInvoices(buyerApplication, callbackFn){
    if(application.isAuthenticated()){
        var user = application.getUser();
        var email = user.getEmails().getItemByID("jimi.hendrix@greendizer.com");
        
        //Instantiate a  collection
        var collection = email.getInvoices().getAll();
        
        collection.populate(function(){                 //Calling the populate() method with no 
            callbackFn(collection.getResources());      //additional parameters will load the
        }, function(response){                          //last 200 resources available.
            //Error occured...
            console.log(response.getStatusCode());
        });
    }
};

User Information

The Buyer and Seller objects

Through the instantiated client, you can access the Buyer and Seller objects. You can then access information such as the first name or the birthday of the current user.

  • Python
  • Php
  • Java
  • Javascript
from greendizer import BuyerClient

#Initialize a BuyerClient instance
user = BuyerClient(oauth_token='oauth_token').user

#Print details about the current user
print user.firstname
print user.lastname
require_once 'Greendizer/BuyerClient.php';

//Initialize a BuyerClient instance
$buyerClient = new Greendizer_BuyerClient(array('token' => $oauth_token));
$user = $buyerClient->getUser();

//Print details about the current user  
echo $user->getFirstName();
echo $user->getLastName();
import com.greendizer.api.client.BuyerClient;
import com.greendizer.api.resource.buyer.Buyer;

//Initialize a BuyerClient instance
Buyer buyer = new BuyerClient("oauth_token").getUser();

//Print details about the current user
System.out.println(buyer.getFirstName());
System.out.println(buyer.getLastName());
application.init(function(state, error){
    if(state){
        //Passing a callback will allow the resource to be loaded from the server
        application.getUser(function(user){
            //Print details about the current user
            console.log(user.getFirstname());
            console.log(user.getLastname());
        });
    }
});

The Company object

The Company object includes the information about the company a specific seller belongs to.

  • Python
  • Php
  • Java
  • Javascript
from greendizer import SellerClient            
            
#Initialize a SellerClient instance
user = SellerClient(oauth_token='oauth_token').user

#Print details about the current user
print user.firstname

#Print details about his company
print user.company.name
require_once 'Greendizer/SellerClient.php';

//Initialize a SellerClient instance
$sellerClient = new Greendizer_SellerClient(array('token' => $oauth_token));
$user = $sellerClient->getUser();

//Print details about their company
echo $user->getCompany()->getName();
import com.greendizer.api.client.SellerClient;
import com.greendizer.api.resource.sellers.Seller;

//Initialize a BuyerClient instance
Seller seller = new SellerClient("oauth_token").getUser();

//Print details about the current user
System.out.println(seller.getFirstName());

//Print details about his company
System.out.println(seller.getCompany().getName());
application.init(function(state, error){
    if(state){
        application.getUser(function(user){
            //Print details about the current user
            console.log(user.getFirstname());
            
            user.getCompany(function(company){
                //Print details about the company
                console.log(company.getName());
            });
        });
    }
});

The Settings object

The Settings object represents the current user settings.

  • Python
  • Php
  • Java
  • Javascript

settings = BuyerClient(oauth_token='oauth_token').user.settings

#Print elements of the user-defined settings
print settings.language
require_once 'Greendizer/BuyerClient.php';

//Initialize a BuyerClient instance
$buyerClient = new Greendizer_BuyerClient(array('token' => $oauth_token));
$user = $buyerClient->getUser();

$settings = $user->getSettings();

//Print elements of the user-defined settings
echo $settings->getLanguage();
import com.greendizer.api.client.BuyerClient;
import com.greendizer.api.resource.buyer.Buyer;
import com.greendizer.api.resource.Settings;

//Initialize a BuyerClient instance
Buyer buyer = new BuyerClient("oauth_token").getUser();

Settings settings = buyer.getSettings();

//Print elements of the user-defined settings
System.out.println(settings.getLanguage());
application.init(function(state, error){
    if(state){
        //A resource can be loaded without retrieving its parents.
        application.getUser().getSettings(function(settings){
            //Print elements of the user-defined settings
            console.log(settings.getLanguage());
        });
    }
});
